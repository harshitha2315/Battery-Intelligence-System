# Battery Intelligence System

## Overview
Battery Intelligence System is an IoT-based Smart Battery Management System developed using ESP32, Blynk Cloud, and Wokwi Simulation. The system continuously monitors battery parameters, detects faults, classifies battery health, and uploads real-time telemetry to the cloud.

## Features
- Real-time battery voltage monitoring
- Battery State of Charge (SOC) estimation
- Health status classification
- Runtime mode detection
- Weak cell detection
- Critical imbalance detection
- Rapid voltage change detection
- Sensor fault detection
- Relay protection system
- Buzzer alarm
- LCD display
- Cloud dashboard using Blynk
- Wokwi simulation

## Hardware
- ESP32
- LCD 16x2 I2C
- Relay Module
- Buzzer
- Potentiometers (Battery Cell Simulation)

## Software
- Arduino IDE
- ESP32 Board Package
- Blynk Cloud
- Wokwi Simulator
