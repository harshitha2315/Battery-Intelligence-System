# Battery Intelligence System

## Overview

Battery Intelligence System is an IoT-based Smart Battery Management System developed using ESP32, Blynk Cloud, and Wokwi Simulator. The system continuously monitors battery cell voltages, detects faults, classifies battery health, and provides real-time cloud monitoring.

## Features

* Real-time battery monitoring
* Pack voltage calculation
* Battery State of Charge (SOC)
* Health status classification
* Runtime mode monitoring
* Risk level indication
* Fault detection and history
* Operator recommendations
* ESP32 cloud connectivity using Blynk
* LCD display and Serial Monitor output

## Hardware Used

* ESP32 Development Board
* LCD I2C Display
* Relay Module
* Buzzer
* Potentiometers (Battery Cell Simulation)

## Software Used

* Arduino IDE
* Wokwi Simulator
* Blynk IoT Cloud
* GitHub
